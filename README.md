# بوت ديسكورد - لعبة تخمين الأرقام 🎮

بوت ديسكورد تفاعلي للعبة تخمين الأرقام بالعربية!

## كيفية اللعب

### بدء لعبة جديدة
```
!لعبة [عدد الخانات]
```
مثال: `!لعبة 3` - لإنشاء رقم مكون من 3 خانات

### التخمين
بعد بدء اللعبة، اكتب الرقم مباشرة:
```
123
```

### إيقاف اللعبة
```
!إيقاف
```

## كيف تعمل اللعبة؟

1. البوت ينشئ رقم عشوائي بعدد الخانات التي اخترتها
2. أنت تحاول تخمين الرقم
3. البوت يخبرك بـ:
   - ✅ عدد الأرقام الصحيحة في المكان الصحيح
   - ⚠️ عدد الأرقام الصحيحة في المكان الخطأ
   - 📝 إجمالي الأرقام الصحيحة

### مثال:
- الرقم السري: `443`
- تخمينك: `534` → رقمان صحيحان في المكان الخطأ (الرقمان 4 و 3)
- تخمينك: `543` → رقمان صحيحان في المكان الصحيح
- تخمينك: `344` → 3 أرقام صحيحة، واحد منهم في المكان الصحيح

## تشغيل البوت

```bash
npm start
```

## المتطلبات

- Node.js
- Discord Bot Token (موجود في ملف .env)

## ملاحظة مهمة 🔒

**تحذير:** التوكن الخاص بالبوت يجب أن يبقى سرياً! لا تشاركه مع أحد أبداً.
إذا تم تسريب التوكن، يجب عليك:
1. الذهاب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. اختيار البوت الخاص بك
3. الذهاب إلى قسم "Bot"
4. الضغط على "Reset Token" لإنشاء توكن جديد
5. تحديث ملف .env بالتوكن الجديد
